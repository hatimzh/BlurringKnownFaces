# BlurringKnownFaces
Python program based on computer vision to bluring detected known faces.

# Usage
- Install opencv and face_recognition:
  
  ```terminal
  pip install opencv-python,face_recognition

or ```terminal 
    python -m pip install opencv-python,face_recognition
    ```
